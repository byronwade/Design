# Design contract changelog

## 1.0.0-alpha.1 — 2026-07-21

- Reorganized the repository around one installable `.design/` contract.
- Preserved Google `DESIGN.md` alpha compatibility and pinned CLI `0.3.0`.
- Added deterministic global → family → platform → surface inheritance.
- Added native iOS, Android, macOS, Windows, GNOME, and KDE Plasma overlays.
- Added web application, web marketing, and Electron composite profiles.
- Added target resolution so agents do not load contradictory sibling verticals.
- Added canonical Agent Skills and generated Codex, Claude, and Copilot adapters.
- Added install, sync, resolve, list, and validate commands.
- Added source-integrity hashes, JSON schemas, unit tests, and GitHub Actions validation.
- Retired duplicated root documentation, token bundles, CSS starters, and duplicated agent-specific Skills in favor of canonical contracts and generated adapters.
