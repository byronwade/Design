# Apple interaction and design principles

A web-oriented reference for fluid, physical interaction design: low-latency feedback, direct manipulation, interruptible motion, velocity-aware springs, momentum, spatial consistency, materials, typography, accessibility, and foundational product-design principles.

The source is divided into three sections:

1. [`apple-interaction-design/01-fluid-interaction-and-springs.md`](apple-interaction-design/01-fluid-interaction-and-springs.md)
2. [`apple-interaction-design/02-gesture-material-accessibility-and-type.md`](apple-interaction-design/02-gesture-material-accessibility-and-type.md)
3. [`apple-interaction-design/03-foundations-process-and-reference.md`](apple-interaction-design/03-foundations-process-and-reference.md)

Use this guidance to extend Calm Paper Workbench interaction behavior. It does not override the canonical system’s layout, density, color, navigation, component, or elevation rules. Glass, translucency, bounce, sound, and haptics remain contextual tools rather than default visual language.
